# Rules Directory

Thu muc `rules/` da duoc chuan hoa theo taxonomy:

- `platform/use_case/telemetry/...`

Vi du:

- `rules/windows/execution/process_creation/...`
- `rules/windows/credential_access/authentication/bruteforce/...`

Quy uoc:

- Sigma YAML: uu tien bo sung metadata van hanh trong truong `x_metadata`
- Native Kibana JSON rule: uu tien bo sung metadata van hanh trong truong `metadata`

Tai lieu chi tiet:

- `docs/rule-architecture.md`
- `schemas/rule_metadata.schema.json`
- `templates/rule_template_sigma.yml`
- `templates/rule_template_threshold.json`
